First Class Hoops

This is an advanced Basketball teaching application that utilizes high-quality instructional videos from some of the most well known coaches and former players to ever play the game!  Videos focus on X's and O's strategy, Shooting Instruction, Leadership/Mentality of the game as well as interpreting and using Statistics.
